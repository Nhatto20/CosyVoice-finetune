# trade_view

A minimal Vite + React (JavaScript) starter project.

Getting started (Windows, cmd.exe):

1. Install dependencies:

   npm install

2. Run development server:

   npm run dev

3. Build for production:

   npm run build

Notes:

- Make sure Node.js and npm are installed. If `npm` is not recognized, install Node.js from https://nodejs.org/ and restart your terminal.
- This project uses Vite as the dev server and bundler.
